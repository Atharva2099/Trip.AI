# Trip.AI — Roadmap & Future Features

| # | Feature | Description | Effort |
|---|---------|-------------|--------|
| 1 | **Tavily Search Grounding** | Real-time search for verifying places have not closed. Search Reddit (r/OffTheBeatenPath, r/travel, r/solotravel), Atlas Obscura for niche hidden gems. Feed results into LLM prompt. | ~2h |
